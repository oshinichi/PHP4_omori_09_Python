## 使い方
　グーグルコラボのため、ランタイムでそのまま実行。
 途中ファイル読み込みが必要であるが、キャンセルでそのまま進む。
　すでに読み込まれたファイルで３つのグループにクラス分けがおこわなれる。
 これを使って、この場合、英語と日本語の相関でクラス分けしているが、
 例えば、英語が得意であるが日本語が不得手の層に対して何らかの対策や場合によっては売り込みをかけるといった効果的な
 対応が可能となるらしい。
